# Flask の実装テスト

## 概要

- Try implementing Flask

## 環境設定

### macOS

- `export FLASK_APP=flaskr`
- `export FLASK_DEBUG=1`

### Windows

- `$env:FLASK_APP=flaskr`
- `$env:FLASK_DEBUG=1`

## 実行

- `flask run`
- `http://127.0.0.1:5000/`
